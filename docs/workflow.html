Weekly Flow / 毎週の流れ

Attendance and Role Assignment / 出欠確認と担当確定

  1. Mon around 09:00 / 月曜09:00ごろ

    Attendance request / 出欠依頼

    The attendance form for that week's practice is emailed to the Google Group. If plans change, submit the same form again; the latest response from the signed-in account is used for attendance. / その週の練習用の出欠フォームをGoogle Groupへメールで送ります。予定が変わった場合は同じフォームから再回答でき、ログイン中のアカウントによる最新の回答が出欠に反映されます。

  2. Thu around 09:00 / 木曜09:00ごろ

    Reminder / リマインド

    A reminder is emailed to the Google Group before the initial response deadline. / 初回回答締切前に、Google Groupへリマインドメールを送ります。

  3. Thu 17:00

    Initial attendance deadline / 初回出欠締切

    The latest response received before 17:00 fixes the role-candidate pool. The Form remains open for attendance changes until 30 minutes before practice. A new Attend response after 17:00 is added to the attendance list but is not included in automatic role selection, and the respondent receives a reminder to answer before the deadline whenever possible. A member who changes to Not attending after 17:00 is removed from the candidate pool. / 17:00より前に届いた最新回答をもとに担当候補を確定します。出欠フォームは練習開始30分前まで引き続き利用できます。17:00以降の新しい参加登録も参加者一覧には反映しますが、自動担当選出の対象には含めず、可能な限り締切前に回答するよう本人へ注意メールを送ります。17:00以降に不参加へ変更した方は担当候補から外します。

  4. Thu from 17:05 / 木曜17:05以降

    Role assignment / 担当確定

    The system selects the setup and return assignees and emails the results to the Google Group, together with any messages submitted for the whole club. At around 17:05, each assignee also receives an individual email. The setup email includes a link to the key pickup confirmation page. If an assignee later changes to Not attending, the Group is notified; that assignee must reply to the Group, find a replacement, and confirm the handoff. The system does not assign a replacement automatically. See How Roles Are Assigned for details. / システムが準備担当と返却担当を選び、全体への連絡事項とともにGoogle Groupへメールで知らせます。17:05ごろには各担当者へ個別メールも届き、準備担当のメールには鍵受け取り確認ページへのリンクが含まれます。担当者が後から不参加へ変更した場合はGoogle Groupへ通知され、その担当者本人がグループへ返信して代役を募集し、引き継ぎを確定します。システムによる代役の自動割当は行いません。詳しくは担当者の決まり方をご覧ください。

  5. Fri 13:00-15:30

    Key pickup / 鍵受け取り

    The setup assignee collects the school key and records the pickup by 15:30 using the confirmation page linked in the individual email. See How Roles Are Assigned for full instructions. / 準備担当は学校の鍵を受け取り、個別メールに記載された確認ページから15:30までに受け取り完了を記録します。詳しい手順は担当者の決まり方をご覧ください。

  6. Fri from 15:30 / 金曜15:30以降

    Safety alert / 未確認アラート

    If pickup has not been confirmed by 15:30, an alert is emailed to the Google Group. Anyone who can collect the key instead should first notify the group and then complete pickup by 16:30. / 15:30までに鍵受け取りの完了記録がない場合は、Google Groupへアラートメールを送ります。代わりに受け取りに行ける方は、まずグループへ周知し、16:30までに受け取りを完了してください。

  7. Fri 18:30 / 金曜18:30

    Self-service change cutoff / 自己変更締切

    The attendance Form closes 30 minutes before practice. After this point, email any attendance change directly to the Google Group. / 練習開始30分前に出欠フォームを閉じます。これ以降の変更はGoogle Groupへ直接メールしてください。

  8. Fri by 18:45

    Equipment / 道具

    The setup assignee brings the club racket bag and portable net to the gym before it opens. See How Roles Are Assigned for details. / 準備担当は、開錠前に部のラケットバッグとポータブルネットを体育館へ運びます。詳しくは担当者の決まり方をご覧ください。

  9. Fri 18:45

    Opening / 開錠

    The setup assignee opens the gym at 18:45. Everyone else is also encouraged to help set up the courts from 18:45 as they arrive. See How Roles Are Assigned for details. / 準備担当が18:45に体育館を開けます。準備担当以外の参加者も、到着したらコート準備にご協力ください。詳しくは担当者の決まり方をご覧ください。

  10. Fri 19:00-21:00

    Practice / 練習

    Indoor gym shoes are required. During practice, everyone helps with the courts, equipment, and cleanup. / 体育館シューズは必須です。コートや道具の準備、練習後の片付けは、参加者全員で協力して行います。

  11. Fri after 21:00

    Locking and key return / 施錠・鍵返却

    After practice, the return assignee locks the gym, returns the key, and takes the equipment or arranges its next handoff. See How Roles Are Assigned for details. / 練習後は、返却担当が体育館を施錠し、鍵を返却して、道具の持ち帰りまたは次回の受け渡しを行います。詳しくは担当者の決まり方をご覧ください。
