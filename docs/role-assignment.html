Assigned Roles / 自動選出する担当

Setup Role and Return Role / 準備担当・返却担当

Setup role / 準備担当

The setup role combines four tasks: collecting the school key from the Mishima Municipal Nishikida Elementary School office between 13:00 and 15:30 on Friday, recording the pickup by 15:30, bringing the club racket bag and portable net, and opening the gym at 18:45. / 準備担当は、金曜13:00から15:30までの三島市立錦田小学校事務室での鍵受け取り、15:30までの完了記録、部のラケットバッグとポータブルネットの運搬、18:45の開錠を担当します。

The setup assignee receives an individual email at around 17:05 on Thursday. After collecting the key, open the confirmation link in that email and press the button by 15:30 on Friday. The Google Group is notified once the pickup is recorded. If no confirmation is recorded by 15:30, the group receives an alert. In an emergency, anyone collecting the key instead should notify the Google Group first and complete pickup by 16:30. / 準備担当には木曜17:05ごろに個別メールが届きます。鍵を受け取ったら、メール内の確認リンクを開き、金曜15:30までにボタンを押して完了を記録してください。記録されるとGoogle Groupへ通知されます。15:30までに記録がない場合は、グループへアラートが送られます。緊急時に代わりに受け取りに行く方は、まずGoogle Groupへ周知し、16:30までに受け取りを完了してください。

At the school office, complete the application form with the date and time, the group name (NIG Badminton Club), and your name. The club racket bag is usually handed over at the entrance to the Animal Facility. At 18:45, unlock the gym, open the doors, and disarm SECOM. / 学校事務室の申込書には、日時、団体名（遺伝研バドミントン部）、氏名を記入します。道具が入った部のラケットバッグは、通常、動物飼育実験棟エントランスで受け渡されます。18:45に体育館を開錠し、扉を開けてセコムを解除します。

Return role / 返却担当

The return role covers closing the gym after practice, turning SECOM on, locking the doors, placing the key in the return box outside the Mishima Municipal Nishikida Elementary School office, and either taking the equipment back or arranging its next handoff. / 返却担当は、練習後の体育館の片付け、セコムのセットと施錠、三島市立錦田小学校事務室前の返却ポストへの鍵返却、道具の持ち帰りまたは次回の受け渡しを担当します。

Support / サポート

If you need help carrying equipment, arranging a handoff, or completing another part of your role, please ask the Google Group for support. / 道具の運搬や受け渡しなどで手助けが必要な場合は、Google Groupでサポートを募ってください。

Candidates / 候補者

Who Is Eligible / 担当候補になる方

  1. 1

    Latest response / 最新回答

    For each email address, the system uses only the most recent Google Form response for that week's practice. / メールアドレスごとに、その週の練習に対する最新のGoogleフォーム回答だけを使用します。

  2. 2

    Attendance / 参加予定

    Only members who selected "Yes, I will attend" are considered for a role. / 「参加します」と回答した部員だけが担当候補になります。

  3. 3

    Role choice / 担当選択

    Members who choose setup, return, or either role become candidates for the corresponding role. A response entered under Other is treated as a reason the member cannot take a role. Because the key confirmation page is restricted to NIG accounts, only attendees responding with an NIG account can be selected for setup. / 準備担当、返却担当、どちらでも可を選んだ方が、それぞれの担当候補になります。「その他」に入力された回答は、今週担当できない理由として扱います。鍵確認ページは遺伝研アカウント限定のため、準備担当は遺伝研アカウントで回答した参加者から選びます。

Selection Rules / 選出ルール

Balancing the Load / 負担が偏らないように選びます

Manual entries / 手動指定

If an administrator names a setup or return assignee in the Practice Sessions sheet, that assignment takes priority only if the person is still eligible based on the latest response. A manually assigned setup person must use an NIG account. / 管理者がPractice Sessionsシートで担当者を指定した場合は、その方が最新回答でも候補条件を満たしているときに限り、手動指定を優先します。準備担当の手動指定には遺伝研アカウントが必要です。

No reshuffle / 再抽選しない

If roles have already been assigned for the same practice, running the process again keeps those assignments as long as each person remains eligible. / 同じ練習日の担当がすでに決まっている場合は、各担当者が引き続き候補条件を満たす限り、処理を再実行しても変更しません。

History / 履歴

Among eligible volunteers, priority goes to people with fewer previous assignments recorded in the Role Assignments sheet. / 候補者の中では、Role Assignmentsシートに記録された過去の担当回数が少ない方を優先します。

Same-week load / 同週の重複

When another eligible volunteer is available, the system avoids assigning both roles to the same person. / 他に候補者がいる場合は、同じ方が準備担当と返却担当を兼ねないようにします。

Role-specific history / 担当別の履歴

If total assignment counts are tied, priority goes to the person who has performed that particular role fewer times. / 担当総数が同じ場合は、その役割の過去の担当回数が少ない方を優先します。

Recency and final tie / 直近担当と最終判定

If the counts are still tied, priority goes to the person whose most recent assignment was longer ago. Any remaining tie is resolved by a consistent ordering unique to that practice date, rather than alphabetically. / それでも回数が同じ場合は、最後の担当からより時間がたっている方を優先します。なお同順位の場合は、氏名順に偏らないよう、その練習日に固有の固定順で決定します。

If no volunteer is available / 候補者がいない場合: The role remains unassigned, and members coordinate through the Google Group. / 担当は未割当となり、Google Groupで対応を相談します。

Notifications / 通知

Notifications Sent at 17:05 on Thursday / 木曜17:05ごろの通知

Google Group email / 全体メール

The Google Group receives the setup and return assignments, an attendance summary, and any messages submitted for the whole club. / Google Groupには、準備担当・返却担当、出欠状況、フォームに入力された全体への連絡事項を送ります。

Individual email / 個別メール

Each assignee receives an individual email. The setup email also contains a key pickup link. After collecting the key, the assignee opens the link and presses the button to record completion by 15:30. / 各担当者には個別メールが届きます。準備担当のメールには鍵受け取り確認リンクも含まれます。鍵を受け取った後にリンクを開き、ボタンを押すと、15:30までの完了記録として保存されます。
