Assigned Roles / 自動選出する担当

Setup Role and Return Role / 準備担当・返却担当

Setup role / 準備担当

Pickup of the school key at the Mishima Municipal Nishikida Elementary School office on Friday between 13:00 and 15:30, key pickup completion recording by 15:30, carrying the club racket bag and portable net, and opening the gym at 18:45 are treated as one setup role. / 金曜13:00-15:30の三島市立錦田小学校事務室での鍵受け取り、15:30までの鍵受け取り完了記録、部のラケットバッグとポータブルネットの運搬、18:45の開錠をまとめて準備担当とします。

The setup role receives an individual assignment email around Thursday 17:05. Use the key pickup confirmation link in that email to record completion by 15:30 on Friday. Once completion is recorded, the Google Group is notified; if it is not recorded by 15:30, an incomplete-pickup alert is sent to the Google Group. / 準備担当には木曜17:05ごろに個別担当メールが届きます。そのメール内の鍵受け取り確認リンクを使って、金曜15:30までに完了を記録してください。完了記録が付くとGoogle Groupへ通知され、15:30までに記録がない場合は未完了アラートがGoogle Groupへ送られます。

On the school application form, write the date/time, group name (NIG Badminton Club), and your name. The club racket bag with equipment is usually handed off at the Animal Facility entrance. Open the gym at 18:45, unlock the doors, and disarm SECOM. / 学校の申込書には日時、団体名（遺伝研バドミントン部）、氏名を記入します。道具が入った部のラケットバッグは、通常、動物飼育実験棟エントランスで受け渡されます。18:45に体育館を開錠し、扉を開けてセコムを解除します。

Return role / 返却担当

After practice, locking the gym, turning SECOM on, placing the key in the return drop post in front of the Mishima Municipal Nishikida Elementary School office, and carrying the equipment back or coordinating the next handoff are treated as one return role. / 練習後の体育館の施錠、セコムON、三島市立錦田小学校事務室前の返却ポストへの鍵返却、道具の持ち帰りまたは次回受け渡し調整をまとめて返却担当とします。

After practice, turn SECOM on, lock the gym, place the key in the return drop post, and bring the equipment back or coordinate the next handoff. / 練習後はセコムをONにして施錠し、鍵を返却ポストへ投函し、道具を持ち帰るか次回の受け渡しを調整します。

Support / サポート

If carrying equipment, handoff, or other role logistics need extra help, ask for support on the Google Group. / 道具の運搬、受け渡し、その他の担当作業でサポートが必要な場合は、Google Groupでヘルプを募ってください。

Candidates / 候補者

Only Current-Week Attendees Are Considered / 今週参加予定の回答だけを使います

  1. 1

    Latest response / 最新回答

    For each email address, only the latest Google Form response for this week's practice is used. / メールアドレスごとに、今週の練習に対する最新のGoogleフォーム回答だけを使います。

  2. 2

    Attendance / 参加予定

    Only people who answered that they will attend are eligible for assignment. / 「参加します」と回答した人だけが担当候補になります。

  3. 3

    Role choice / 担当選択

    People who choose setup, return, or either role are candidates for the corresponding role. If Other is used to explain why a role is difficult this week, that response is not treated as a volunteer. / 準備担当、返却担当、どちらでも可を選んだ人が対応する担当の候補になります。「その他」で今週担当が難しい理由を書いた回答は、担当可能者としては扱いません。

Selection Rules / 選出ルール

Balancing the Load / 負担が偏らないように選びます

Manual entries / 手動指定

If a maintainer has already entered a setup or return name in the Practice Sessions sheet, that manual value is kept. / 管理者がPractice Sessionsシートに準備担当または返却担当を手動入力している場合、その指定を優先します。

No reshuffle / 再抽選しない

If assignments for the same practice already exist, rerunning the script keeps them unless the manual value has changed. / 同じ練習日の担当がすでに確定している場合、手動指定が変わらない限り再実行しても担当は入れ替えません。

History / 履歴

Among volunteers, people with fewer past assignments in the Role Assignments sheet are ranked earlier. / 担当可能者の中では、Role Assignmentsシート上の過去担当回数が少ない人を優先します。

Same-week load / 同週の重複

The script avoids giving both roles to the same person when another eligible volunteer is available. / 他に候補者がいる場合、同じ人に準備担当と返却担当の両方が集中しないようにします。

If no volunteer is available / 候補者がいない場合: The role is left as Unassigned, and members should coordinate support through the Google Group. / その担当は未割当になり、Google Groupでサポートや交代を調整します。

Notifications / 通知

What Is Sent at Thursday 17:05 / 木曜17:05に送られるもの

Google Group email / 全体メール

The Google Group receives the confirmed setup role, return role, attendance summary, and any shared messages from the attendance form. / Google Groupには、確定した準備担当・返却担当、参加状況、出欠フォームの全体連絡欄の内容が送られます。

Individual email / 個別メール

Assigned members receive an individual email. The setup role assignee also receives the key pickup completion link used to record pickup by 15:30. / 担当者には個別メールが届きます。準備担当には、15:30までに鍵受け取り完了を記録するためのリンクも届きます。
