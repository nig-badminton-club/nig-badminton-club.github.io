Source of Truth / 予定の入力場所

Practice Sessions Sheet / Practice Sessionsシート

Maintainers add new practice dates in the private admin spreadsheet, on the Practice Sessions sheet. The public repository does not need to be edited for ordinary schedule additions. / 管理者は非公開の管理スプレッドシートのPractice Sessionsシートに新しい練習日を追加します。通常の予定追加では、公開リポジトリを編集する必要はありません。

Row Format / 行の書き方

Fields to Fill / 入力する列

Date / 日付

Use YYYY-MM-DD for session_id and date, for example 2026-07-10. / session_id と date は 2026-07-10 のように YYYY-MM-DD で入力します。

Time and place / 時刻・場所

Fill start_time, end_time, and location when they differ from the usual 19:00-21:00 at Mishima Municipal Nishikida Elementary School Gym. / 通常の三島市立錦田小学校体育館19:00-21:00と異なる場合は、start_time、end_time、location を入力します。

Status / ステータス

Use scheduled for confirmed practices, tentative for provisional dates, and cancelled for cancelled practices. / 確定した練習は scheduled、仮予定は tentative、中止は cancelled にします。

Optional note / 任意メモ

Use public_note for a note that may appear in the public schedule. Keep private details out of this field. / 公開してよい連絡事項は public_note に入れます。非公開情報は入れないでください。

Automatic Updates / 自動反映

What Updates After a Row Is Added / 行を足したあとに更新されるもの

  1. Daily

    Refresh / 定期更新

    The automation refreshes the form, calendar, public summary, and member count once a day. / 自動処理が1日1回、フォーム、カレンダー、公開サマリ、部員数を更新します。

  2. Calendar

    Calendar event / カレンダー予定

    Scheduled practice rows are created or updated as Google Calendar events. / scheduled の練習行はGoogleカレンダーの予定として作成または更新されます。

  3. Form

    Attendance form / 出欠フォーム

    The attendance form is updated to the practice in the current week. Future practices are shown on the website and calendar, but are not selectable in the form until their week. / 出欠フォームは今週の練習だけを対象に更新されます。将来の練習はウェブサイトとカレンダーには表示されますが、その週になるまでフォームの選択肢には出ません。

  4. Website

    Public schedule / 公開予定

    The website reads the public summary and displays upcoming practices only. / ウェブサイトは公開サマリを読み込み、今後の練習予定だけを表示します。

Manual Refresh / 手動更新

When Immediate Reflection Is Needed / すぐ反映したい場合

Run refresh / 更新関数を実行

After editing the sheet, maintainers can run refreshFormCalendarAndSummary from Apps Script to update the form, calendar, and public summary immediately. / シート編集後すぐに反映したい場合、管理者はApps Scriptで refreshFormCalendarAndSummary を実行します。

Cancelled dates / 中止日の扱い

If a date that already has a calendar event is later cancelled, check the Google Calendar event manually as part of the cancellation notice. / すでにカレンダー予定が作成された日を後から中止にする場合は、中止連絡の一環としてGoogleカレンダー予定も手動で確認してください。
